Data#

The library provides a python client for the Financial Modelling Prep API. To use the client one needs to provide the API key aither directly to the client or via the FMP_API_KEY environment variable. The API offers 1 minute, 5 minutes, 15 minutes, 30 minutes, 1 hour and daily historical prices.

from quantflow.data.fmp import FMP
cli = FMP()
cli.url

'https://financialmodelingprep.com/api'

stock = "KNOS.L"
