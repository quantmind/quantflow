analytical_cdf(t: ndarray[tuple[Any, ...], dtype[floating[Any]]] | float, n: ndarray[tuple[Any, ...], dtype[floating[Any]]] | float) → ndarray[tuple[Any, ...], dtype[floating[Any]]] | float#

CDF of the number of events at time t.

It’s given by

(80)#\[F\left(n\right)=\frac{\Gamma\left(\left\lfloor n+1\right\rfloor ,\lambda\right)}{\left\lfloor n\right\rfloor !}\]

where \(\Gamma\) is the upper incomplete gamma function.

analytical_mean(t: ndarray[tuple[Any, ...], dtype[floating[Any]]] | float) → ndarray[tuple[Any, ...], dtype[floating[Any]]] | float#

Expected value at a time horizon

analytical_pdf(t: ndarray[tuple[Any, ...], dtype[floating[Any]]] | float, n: ndarray[tuple[Any, ...], dtype[floating[Any]]] | float) → ndarray[tuple[Any, ...], dtype[floating[Any]]] | float#

Probability density function of the number of events at time t.

It’s given by

(81)#\[f\left(n\right)=\frac{\lambda^{n}e^{-\lambda}}{n!}\]
analytical_variance(t: ndarray[tuple[Any, ...], dtype[floating[Any]]] | float) → ndarray[tuple[Any, ...], dtype[floating[Any]]] | float#

Expected variance at a time horizon

Jacobian of the CDF

It’s given by

(82)#\[\frac{\partial F}{\partial\lambda}=-\frac{\lambda^{\left\lfloor n\right\rfloor }e^{-\lambda}}{\left\lfloor n\right\rfloor !}\]
